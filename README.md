### 使用方式

如服务器实行邀请制，在游戏中使用：

`/invite <Username>` - 邀请一个指定玩家

`/whoinvite <Username>` - 查看指定玩家由谁邀请

`/ireload` - 重载插件

### 示例配置文件
```
  modakeWuPo: modakeWuPo 
  redmoon5428: modakeWuPo 
  <被邀请人>: <邀请人>
```
